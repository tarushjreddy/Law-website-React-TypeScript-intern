# Zytcode-AlMuehi

## Technologies and Tools

- TextEditor - Visual Studio Code
- VSCode Extensions
  - `Prettier - for code formatting`
  - `ES7 React/Redux/GraphQL/React-Native snippets`
- React
- TypeScript
- Material UI

## Usage

```cmd
git clone REPO_ADDRESS

cd FOLDER_NAME

git checkout dev

yarn install

yarn start
```
